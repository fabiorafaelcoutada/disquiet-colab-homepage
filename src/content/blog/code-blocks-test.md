## Python Code
```python
def hello_world(name):
  """Prints a greeting."""
  print(f"Hello, {name}!")
hello_world("Fabio")
```

## JavaScript Code
```javascript

  console.log(`Hello, ${name}!`);
}
greet("Fabio");
```
